About

Global warming is the long-term heating of Earth's surface observed since the pre-industrial period (between 1850 and 1900) due to human activities, primarily fossil fuel burning, which increases heat-trapping greenhouse gas levels in Earth's atmosphere. This term is not interchangeable with the term "climate change".So In this project I had made the analysis and visualization over the impact of Global warming using the Visual Studio Code Software and Python Language and made the UI with Streamlit I had used the Pandas, numpy and Plotly library.
